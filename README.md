# Multi-Cloud Cost Optimization & Resilience Framework

## Features

- Multi-cloud deployment (AWS + Azure)
- Automatic failover system
- Cost-based decision making (simulated)
- Logging and monitoring
- Graph-based analysis

## Technologies Used

- Python
- Flask
- AWS EC2
- Azure VM
- Matplotlib

## How to Run

1. Deploy flask_app.py on AWS & Azure
2. Run main.py locally
3. Generate logs
4. Run graph.py for visualization
